Команда:"Превысокомногорассмотрительствующие Охламоны"
 
    Литературный помощник (RAG AI Service)

Интеллектуальный сервис для поиска по книгам и ответов на вопросы на основе ИИ (GigaChat). Проект разработан для отборочного этапа направления «Искусственный интеллект».

    Основные возможности
- Умный поиск (RAG) Бот анализирует содержимое .txt файлов и находит точные ответы.
- Работа с цитатами: Ответы подкрепляются прямыми выдержками из текста (требование кейса).
- Интерактив: Автоматическая генерация викторин, пересказов и интересных фактов.
- Гибкость: Возможность загрузки собственных произведений пользователем.

    Стек технологий
- Язык: Python 3.12.3
- Интерфейс: Telegram Bot API (Aiogram 3.x)
- Нейросеть: GigaChat API
- Метод: Контекстный поиск (Retrieval-Augmented Generation)

    Инструкция по запуску
1. Склонируйте репозиторий: "git clone [ссылка]"
2. Установите зависимости: "pip install -r requirements.txt"
3. Создайте файл ".env" в корневой папке и добавьте свои API-ключи (см. ".env.example").
4. Поместите книги в формате ".txt" в папку "books/".
5. Запустите бота: "python main.pу"

    Управление библиотекой
- Предустановленные книги: Находятся в папке "books/" и доступны по номерам 1–14.
- Добавление своих книг: Пришлите боту файл в формате ".txt". Система автоматически сохранит его, проиндексирует и предложит перейти к анализу.
- Масштабируемость: Архитектура позволяет обрабатывать тексты объемом до 25 000 символов за один запрос (лимит контекстного окна GigaChat).
    
    Примеры запросов для проверки
- *Поиск факта*: "Найди фрагмент, где описывается дуэль Гринева и Швабрина".
- *Вопрос по сюжету*: "В чем заключался конфликт Базарова и Павла Петровича?".
- *Цитаты*: Бот автоматически выделит ключевые фразы из текста в ответе.
